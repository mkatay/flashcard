# Docker

A Docker egy eszköz, amivel egy alkalmazást becsomagolhatsz minden szükséges környezetével együtt, és így bárhol ugyanúgy fog futni.

Nem csak a programot viszed magaddal, hanem:
- az operációs rendszer egy kis részét
- a szükséges futtatókörnyezetet (pl. Node.js, PHP)
- a beállításokat
- a függőségeket

Ezt a csomagot hívjuk **konténernek**.
