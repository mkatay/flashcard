## ES6 (ECMAScript 2015)

Az ES6 volt a JavaScript legnagyobb ugrása, amely modern, jól strukturált kódolást tett lehetővé.

**Fő újítások ES6-ban:**

- let és const – biztonságosabb változókezelés
- Arrow functionök (()=>) – rövidebb, tisztább kód
- Class szintaxis – objektumorientált programozás
- Modules (import / export) – kód újrahasznosítás
- Template stringek (`Hello ${név}`)
- Destructuring – egyszerűbb adatkinyerés
  
**Miért fontos a gyakorlatban?**

- Könnyebben olvasható, karbantartható kód
- Modern frontend keretrendszerek (React, Vue, Angular) ES6-ra épülnek

Ma már ipari alapkövetelmény az ES6 ismerete.
