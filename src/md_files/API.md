### API (Application Programming Interface)
- Egy általános **programozási felület**, amely lehetővé teszi, hogy két alkalmazás vagy rendszer kommunikáljon egymással.
- Nem kötött semmilyen protokollhoz vagy architektúrához.
- Lehet például:  
  - fájl alapú  
  - könyvtárak vagy SDK-k formájában  
  - webes API (HTTP alapú)

### REST API (Representational State Transfer API)
- Az API egy speciális típusa, amely **HTTP protokollt használ**.
- Követi a **REST architektúra elveit**, mint például:  
  - erőforrások URI-kon keresztül  
  - stateless kommunikáció  
  - szabványos HTTP metódusok (`GET`, `POST`, `PUT`, `DELETE`)
- Leggyakrabban JSON formátumban küldi és fogadja az adatokat.

**A REST API egy speciális, webes API típus, amely HTTP és REST szabályokat használ az adatcseréhez.**
